  1. Some x are not-y.
  2. All x are not-y.
  3. Some x are y, and some are not-y.
  4. No x exist.
  5. Some x exist.
  6. No x are not-y.
  7. Some x are not-y, and some x exist.

Taking x="judges"; y="just";

  1. No judges are just.
  2. Some judges are unjust.
  3. All judges are just.

Taking x="plums"; y="wholesome";

  1. Some plums are wholesome.
  2. There are no wholesome plums.
